Instruct your visitors to choose their country code when entering their mobile number to ensure accurate and correctly formatted data submissions.

== Description ==
The Smart Phone field plugin enables you to create a country drop-down list with country flags. The tag field automatically populates country names in the standard drop-down field in the phone field.

== Features ==
* Live/Frontend validation.
* Automatic country select with IP address (GeoIP).
* Default country selection.
* Exclude countries.
* Only include countries.
* Multiple phone field.
* Country search
* More coming soon...

= Requirements =
* PHP 7.4
* WPForms 1.8.0 or higher
* WooCommerce
* Contact Form 7
* Fluent Forms 
* Elementor Forms

== Frequently Asked Questions ==

= Which plugins are supported by this smart phone field plugin? =
Currently, WPForms, Contact Form 7, Fluent Forms, Elementor Forms and WooCommerce are supported. However, we are working to add support for more plugins, such as Formidable, Ninja Forms, and others.

= Does this plugin send or save numbers with the country code? =
Yes, it saves the number in entries and sends emails with the country code included.

= Does it work with the auto-detect country flag? =
Yes, it works in all popular browsers.


== Installation ==
 - In your WordPress admin dashboard, go to **Plugins >> Add New**.
 - Search for "Smart Phone Field For WPForms" and install the plugin.
 - Alternatively, you can install the plugin manually by uploading the unzipped plugin folder to the **/wp-content/plugins**
